Prefetch Files
LOCATION
C:\Windows\Prefetch\*.pf
Records execution timestamp, run count (up to 8 timestamps), and all files/directories accessed within 10 seconds of launch. Covers the last 128 unique executables on most Windows installs.
PROVESBinary executed, when, how many times
CAVEATDisabled by default on Windows Server

EXECUTION Windows
Shimcache (AppCompatCache)
LOCATION
HKLM\SYSTEM\CurrentControlSet\Control\Session Manager\AppCompatCache
Registry artifact storing metadata for executables the OS has seen. Survives reboots; data written to registry on shutdown. Does not require the binary to have been executed — presence proves the file was observed by the OS.
PROVESBinary path seen by OS, last modification time
CAVEATEntry does NOT guarantee execution occurred
CAPACITY1,024 entries (Win10+), FIFO

LOCATION
C:\Windows\AppCompat\Programs\Amcache.hve

LOCATION

System Resource Usage Monitor retains 30–60 days of per-application network usage (bytes sent/received), CPU time, and energy data. Survives log clearing and records data for every application that touched the network.

LNK Files & Jump Lists
LOCATION
%APPDATA%\Microsoft\Windows\Recent\*.lnk
%APPDATA%\Microsoft\Windows\Recent\AutomaticDestinations\
Automatically created when a user opens a file. Contains MAC timestamps, volume serial number, drive type, original file path, and — critically — the NetBIOS name and MAC address of the source machine if the file was on a remote share.

BONUSEmbeds source host's MAC addr for remote files
SURVIVESFile and share deletion

EXECUTION Linux
bash_history & wtmp
LOCATION
~/.bash_history  |  /var/log/wtmp  |  /var/log/auth.log
bash_history records interactive commands per user; wtmp/btmp record login/logout events in binary format. Both are trivially wiped, but gaps in wtmp timestamps and auth.log residuals often reveal the deletion itself.
PROVESCommands run, login times, source IPs
CAVEATTrivial to clear; gaps are themselves evidence
COMPANIONjournald logs often survive bash_history wipe
PARSERS
last / lastb utmpdump log2timeline Autopsy
